## Billing History Assignment

Create a billing history table using Angular and Tailwind CSS. The idea behind this exercise is to test your knowledge and understanding of Angular as well as other modern front-end technologies.

_NOTE_: The `Bonus Features` is exactly what it sounds like; nothing you are required to do, but are nice to haves.

### Requirements

The dashboard should display transaction history with the following features:

1. **Main Dashboard View**

   - Display transactions in a responsive table or grid
   - Implement sorting by date
   - Show loading states while data is being fetched
   - Implement error handling

2. **Transaction Details**

   - Create a detail view for individual transactions
   - Use route parameters to display transaction details
   - Show loading state while fetching details

3. **Filtering and Search**

   - Add filters for status and category
   - Implement a search function for transaction descriptions
   - Use RxJS operators for debouncing search input

4. **Bonus Features (nice to haves)**

   - Add pagination
   - Implement export functionality
   - Implement delete functionality

### Technical Requirements

- Use Angular
- Style with Tailwind CSS
- Implement proper TypeScript types
- Use RxJS operators for data handling
- Demonstrate Promise usage with async/await
- Include error boundaries
- Add at least one unit test
